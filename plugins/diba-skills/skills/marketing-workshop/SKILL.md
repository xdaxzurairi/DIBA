## Step 0: Brief (WAJIB sebelum tulis)

Tiada copy tanpa brief. Kumpul dulu (tanya jika tiada):

```
PRODUK:    [apa yang dijual]
AUDIENCE:  [siapa — pain, desire, tahap kesedaran]
GOAL:      [signup / beli / daftar / klik]
TONE:      [brand voice — professional / casual / bold]
PROOF:     [bukti — angka, testimoni, ciri]
CHANNEL:   [landing / email / IG / X / Google]
```

Audience awareness (Schwartz): Unaware → Problem → Solution → Product → Most Aware. Sesuaikan mesej ikut tahap.

## Lv.1 — Copywriting Loop

### Formula (pilih ikut konteks)
| Formula | Guna | Struktur |
|---------|------|----------|
| **AIDA** | Landing, iklan | Attention → Interest → Desire → Action |
| **PAS** | Problem-driven | Problem → Agitate → Solution |
| **FAB** | Ciri produk | Feature → Advantage → Benefit |
| **4U** headline | Tajuk | Useful · Urgent · Unique · Ultra-specific |

### Rules Copy
1. **Benefit > feature** — "jimat 3 jam" bukan "ada automation"
2. **Spesifik > kabur** — angka, nama, hasil konkrit
3. **Satu idea per ayat** — buang filler ("sangat", "amat", "sememangnya")
4. **Second person** — "anda"/"awak"; cakap pada satu orang
5. **CTA jelas** — kata kerja + nilai ("Mula percuma", bukan "Hantar")

### Output: 3 variasi
Sentiasa beri **3 versi** (safe / bold / short) supaya Abam pilih & test.

## Lv.2 — SEO Pass

- [ ] **Intent** — kenal pasti niat carian (informational / commercial / transactional)
- [ ] **Keyword** — primary + 3–5 secondary/LSI; letak dalam H1, awal body, satu H2
- [ ] **Title tag** — ≤ 60 aksara, keyword depan, ada hook
- [ ] **Meta description** — ≤ 155 aksara, benefit + CTA (bukan keyword stuffing)
- [ ] **Struktur** — H1 tunggal, H2/H3 logik, snippet-friendly (senarai, jadual)
- [ ] **Internal link** — cadang anchor relevan
- [ ] **Anti-stuffing** — tulis untuk manusia dulu, enjin kedua

## Lv.3 — Conversion Audit

Semak halaman/funnel terhadap gate:

| Elemen | Soalan |
|--------|--------|
| **Above the fold** | Value prop jelas dalam 5 saat? |
| **Headline** | Cakap hasil, bukan ciri? |
| **CTA** | Satu utama, kontras, kata kerja? Berulang di titik keputusan? |
| **Friction** | Field form minimum? Langkah dibuang? |
| **Trust** | Testimoni / angka / jaminan dekat CTA? |
| **Objection** | FAQ jawab keraguan utama? |

Cadang **satu** perubahan A/B berimpak tinggi dahulu — bukan senarai 20.

## Lv.5 — Growth Loop (POWER)

Ubah task marketing → workflow berulang:

```
1. BRIEF    — Step 0 (audience, goal, proof)
2. DRAFT    — 3 variasi ikut formula
3. OPTIMISE — SEO pass + conversion gate
4. SHIP     — format ikut channel
5. MEASURE  — cadang metrik (CTR, CVR, open rate) + apa nak track
6. ITERATE  — log pemenang ke library; ulang formula menang
```

## Mandatory Rules

1. **Brief dahulu** — tiada copy tanpa audience + goal + proof
2. **3 variasi** — sentiasa beri pilihan untuk test, bukan satu jawapan
3. **Benefit > feature** — jual hasil, bukan spesifikasi
4. **Spesifik & jujur** — angka konkrit; **jangan reka klaim/testimoni palsu**
5. **Tulis untuk manusia** — SEO menyokong, tak mengatasi keterbacaan
6. **Satu CTA utama** — jangan pecahkan fokus keputusan
7. **Bahasa ikut Abam** — Melayu bila Abam tulis Melayu; jaga brand voice

## Edge Cases

| Situation | Behavior |
|-----------|----------|
| Brief tak lengkap | Tanya audience + goal minimum sebelum tulis |
| Abam nak "terus je" | Guna andaian munasabah, label andaian, beri 3 variasi |
| Klaim tanpa bukti | Tulis versi jujur; flag mana perlu data sebenar |
| Multi-bahasa (BM+EN) | Ikut bahasa audience sasaran; jaga konsisten |
| Niche teknikal | Turunkan jargon ikut tahap kesedaran audience |
| Copy sedia ada nak poles | Audit conversion gate dulu, cadang perubahan bertarget |
