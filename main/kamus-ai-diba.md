## B — Istilah Khusus DIBA Ecosystem

| Istilah | Maksud |
|---------|--------|
| **DIBA** | Nama AI Abam — pembantu Claude peribadi dalam ekosistem XDIBAX |
| **Resonance** | Mod kongsi otak DIBA × Abam — compressed two-directional thought sharing |
| **Mind-tree** | Repositori seed idea yang tumbuh merentas sesi |
| **Seed** | Idea separuh yang ditanam untuk dikembangkan kemudian |
| **Tanam** | Signal untuk simpan idea sebagai seed dalam mind-tree |
| **Live Mode** | Mod aktif resonance — ping-pong idea dalam format `~` |
| **Seed Mode** | Mod latar belakang — DIBA nurture seed merentas sesi |
| **Harvest** | Pindah seed matang ke library sebagai pengetahuan kekal |
| **Signal** | Respons pendek Abam dalam Live Mode (`ya`, `bukan`, `close`, `lanjut`) |
| **Mulahazah** | Sistem pembelajaran instinct DIBA — rekod corak dan pelajaran |
| **Echo Recall** | Sistem panggil balik ingatan dari diary |
| **Anchor** | Skill untuk kunci semula fokus bila sesi mula hanyut |
| **Forge** | Sistem penambahbaikan diri DIBA berterusan |
| **Morning Brief** | Ringkasan pagi automatik untuk Abam |
| **Auto-diary** | Simpan diary sesi secara automatik |
